# 🔍 VNR VJIET Lost & Found Portal

A modern, highly responsive, and premium web application designed exclusively for the VNR VJIET campus community. This portal allows students, faculty, and administration to seamlessly report, track, and claim lost or found items across the college premises.

Featuring a stunning dark-mode aesthetic, interactive campus zone mapping, and dynamic visual filtering, this platform streamlines the retrieval process and fosters a helpful campus environment.

---

## ✨ Key Features

- **🎓 Campus-Specific Branding:** Fully customized for the VNR VJIET community.
- **🌒 Premium Dark UI/UX:** Built with modern design principles utilizing harmonious dark themes, glassmorphism elements, micro-animations, and premium typography (`Syne` & `DM Sans`).
- **🗺️ Interactive Campus Map:** A unified grid system mapping out various campus zones (e.g., Canteen, Library, Labs). Users can visually filter and locate where items were lost or found.
- **📋 Smart Dashboard & Filters:** Instantly toggle between "Lost", "Found", and "Resolved" items via a highly responsive grid layout. Includes a dedicated search bar for rapid discovery.
- **🤖 AI-Powered Match Badges:** Smart tagging logic highlights potential matches between newly reported lost items and recently found items.
- **📝 Seamless Reporting Form:** A dedicated, easy-to-use interface for users to report items, requiring exact details (category, location, date, and description).
- **🔔 Live Alerts & Notifications:** Real-time pulse alerts and notification badges to keep users informed about recent matches or system announcements.
- **👨‍🏫 Professor/Security Hand-off:** A dedicated directory that showcases items successfully turned over to respective department faculty or campus security, ensuring accountability.

---

## 🛠️ Technology Stack

This interactive prototype/frontend is built purely with modern web standards, requiring no heavy frameworks to load:

- **HTML5:** Semantic, accessible structure.
- **Vanilla CSS3:** Custom design system utilizing CSS variables (custom properties), flexbox/grid layouts, scroll-snapping, and keyframe micro-animations without relying on external libraries.
- **Vanilla JavaScript:** DOM manipulation, smart filtering, map zone interactivity, and form state handling.
- **Google Fonts:** `Syne` (Headings/Display) & `DM Sans` (Body text).
